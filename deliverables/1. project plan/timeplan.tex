The project will be conducted between March and June and is structured into several phases corresponding to the course milestones and deliverables.

\subsection{Phase 1: Project Setup and Planning (March 9 -- March 19)}

\begin{itemize}
\item Kick-off session and first meeting with data coaches
\item Definition of the project objective and research question
\item Initial exploration of available datasets
\item Discussion of potential modelling approaches
\item Preparation and submission of the project plan
\end{itemize}

\subsection{Phase 2: Methodology Development (March 20 -- March 29)}

\begin{itemize}
\item Data collection and preprocessing
\item Selection of relevant spatial and spectral features
\item Design of the machine learning pipeline
\item Refinement of the project plan based on feedback
\end{itemize}

\subsection{Phase 3: Data Processing and Initial Modelling (March 30 -- April 21)}

\begin{itemize}
\item Implementation of the preprocessing pipeline
\item Integration of Sentinel-2 imagery and OpenStreetMap features
\item Training of initial machine learning models
\item Preliminary evaluation of model performance
\item Preparation of the intermediate report and presentation slides
\end{itemize}

\subsection{Phase 4: Model Improvement and Validation (April 22 -- June 6)}

\begin{itemize}
\item Participation in intermediate consultations
\item Refinement of feature engineering and modelling approach
\item Hyperparameter tuning and model validation
\item Error analysis and improvement of prediction results
\item Sparring group meetings and feedback integration
\item Preparation of the draft final report and slides
\end{itemize}

\subsection{Phase 5: Finalization and Presentation (June 7 -- June 28)}

\begin{itemize}
\item Final model improvements and validation
\item Preparation of the final presentation
\item Incorporation of feedback into the report
\item Completion and submission of the final report
\item Submission of sparring group meeting minutes
\end{itemize}
